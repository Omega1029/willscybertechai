# Vite React TypeScript Starter

## Overview
A React TypeScript application built with Vite, featuring Tailwind CSS for styling. The app includes integrations with Supabase, EmailJS, OpenAI, and Stripe.

## Project Structure
- `src/` - Source code
  - `api/` - API related code
  - `components/` - React components
  - `hooks/` - Custom React hooks
  - `lib/` - Library utilities
  - `pages/` - Page components
  - `App.tsx` - Main application component
  - `main.tsx` - Entry point
- `public/` - Static assets
- `supabase/` - Supabase configuration

## Tech Stack
- React 18 with TypeScript
- Vite as build tool
- Tailwind CSS for styling
- React Router for navigation
- Supabase for backend
- EmailJS for email functionality
- OpenAI integration
- Lucide React for icons

## Running the Application
- Development: `npm run dev` (runs on port 5000)
- Build: `npm run build`
- Preview: `npm run preview`

## Configuration
The Vite dev server is configured to:
- Run on port 5000
- Bind to 0.0.0.0 for Replit compatibility
- Allow all hosts for proxy compatibility
